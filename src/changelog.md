# Changelog

## 2026-06-08

- Construcción inicial del prototipo interactivo IA Orienta.
- Creación de SPA estática con HTML5, CSS3 y JavaScript Vanilla.
- Incorporación de pantallas para inicio, perfil, diagnóstico, recomendador, explorador, prompts, aprendizaje y favoritos.
- Centralización de datos simulados en `src/js/data.js`.
- Adición de `.gitignore` para artefactos locales sin ocultar entregables académicos.
- Actualización de documentación para ejecución local y despliegue en GitHub Pages.
- Ajuste de ortografía, menú lateral y copiado real de prompts.
- Documentación del despliegue por GitHub Actions publicando `./src` como artefacto de Pages.
- Traslado del changelog a `src/changelog.md` para mantener la documentación junto al prototipo publicado.
- Mejora responsive con enfoque mobile first para evitar desbordes, tarjetas fuera de pantalla y navegación incómoda en móviles.
- Refuerzo del eje de alfabetización IA en Home, Centro de Aprendizaje, Recomendador, Explorador, Biblioteca de Prompts y Favoritos.
- Adición de sección "¿Qué aprenderás?", cápsulas educativas y explicaciones sobre selección de herramientas, prompts, validación, riesgos y buenas prácticas.
- Cambio de navegación móvil a sidebar lateral abrible/cerrable con overlay, botón de cierre y soporte para Escape.
- Ajuste del sidebar de escritorio para permanecer fijo durante el scroll sin recortarse visualmente.
- Fijación de la barra superior en móvil para mantener visible el título y la navegación durante el scroll.
- Incorporación de botón de volver con historial interno de vistas del prototipo.
